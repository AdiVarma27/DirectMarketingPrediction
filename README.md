# DirectMarketingPrediction
End to end Machine Learning pipeline for Direct Marketing Prediction.

Check out [Modeling & Evaluation](https://github.com/AdiVarma27/DirectMarketingPrediction/blob/master/DirectMarketingPrediction.ipynb) & [Exploratory Data Analysis](https://github.com/AdiVarma27/DirectMarketingPrediction/blob/master/EDA.ipynb) notebooks !
